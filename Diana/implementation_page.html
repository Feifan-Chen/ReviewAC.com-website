General Information

In order to create my web-pages I used the following versions of HTML and CSS:

  • HTML5
  • CSS 3

Also, I used an online real-time HTML editor , because this method was easier than using a normal text editor, as my HTML file was updating in real time. I used the following operating systems:

  • Raspbian (a Debian-based Linux distribution) on the my Raspberry Pi 3
  • Windows 10 on my personal laptop, for working remotely, editing directly on GitLab

As a browser, I used Microsoft Edge on my personal computer for accesing GitLab. For checking the correctness of my program I used an online validator.
