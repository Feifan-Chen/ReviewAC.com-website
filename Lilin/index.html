What is HTML

Short for "HyperText Markup Language", the authoring language used to create documents on the World Wide Web.

History of HTML

  • HTML 3.2 --1997
  • HTML 4.01 --1999
  • XHTML --2000
  • HTML5 --2014

What is CSS

Stands for "Cascading Style Sheet." CSSs are used to format the layout of Web pages. They can be used to define text styles, table sizes, and other aspects of Web pages that previously could only be defined in a page's HTML.

History of CSS

  • CSS1 --1996
  • CSS2 --1998
  • CSS2.1 --2011
  • CSS3 --2001, 2011
